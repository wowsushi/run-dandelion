# run-dandelion
2D橫向捲軸小遊戲，玩家須操縱普公瑛閃躲障礙物，直到倒數時間結束

## 使用技術
- [Phaser.js](https://phaser.io/)
- HTML
- CSS
- JavaScript

## 功能
- 可以操縱普公瑛前進、後退與跳躍
- 挑戰成功會跳出恭喜視窗
- 挑戰失敗可以重新開始新遊戲
